backendANDai – learning journal
================================

This repo is my personal space to **practice, learn, and experiment** with two things I care a lot about:

- **Backend development**
- **AI / LLM-powered development**

I’ll be pushing in all the small projects, spikes, experiments, and notes I do as I try to level up as an engineer. Think of it as a living notebook rather than a polished product.

## What this repo is about

- **Deliberate practice**: building and breaking small things to understand them properly.
- **Backend depth**: databases, schema design, APIs, performance, testing, reliability.
- **AI depth**: LLMs, tools, agents, retrieval, and how they plug into real backends.
- **Learning in public (sort of)**: date-wise logs of what I tried and learned.

If you’re reading this in the future, this repo should tell the story of how my backend + AI skills evolved over time.

## Areas of focus

- **Backend Dev**
  - TypeScript, Node.js, Express
  - PostgreSQL, schema design, indexing
  - Drizzle ORM, migrations, seeding, diagnostics
  - API design, error handling, structure & patterns

- **AI Dev**
  - LLM tooling and agents
  - Connecting LLMs to real data and backends
  - Experimenting with prompts, workflows, and automation

## Tech stack (so far)

This will evolve over time, but currently the project is built around:

- **Language**: TypeScript
- **Runtime**: Node.js
- **Web framework**: Express
- **Database**: PostgreSQL
- **ORM / Migrations**: Drizzle ORM + drizzle-kit
